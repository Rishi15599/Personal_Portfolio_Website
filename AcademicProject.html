IMPLEMENTATION OF SMART DISPLAY USING RASPBERRY PI
A smart display is capable of displaying time, date, weather of the city, the latest update of news, and headlines. it is to provide more information with less interaction to get it. this idea of making the home smarter and saving time.
The main goal of the smart display is to develop a display which looks like a regular display but would have screen inside and user can interact with it using voice command and gestures to receive all the information that could affect how they prepare for the day. By building these features into a display, which people are already using in their morning routine, it is possible to present this information to them in such a way that this information will blend seamlessly together with the task of morning grooming.
